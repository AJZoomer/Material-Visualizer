# Material Visualiser

This is a small JavaFX app I’m building to make Litematica material lists easier to read.  
Instead of scrolling through a giant text file, this shows everything in a clean table with proper counts and stack calculations.

## What it does
- Loads a Litematica material list
- Shows all blocks in a simple UI
- Calculates stacks for you
- Lets you sort and organise stuff properly

## Why I made it
I got sick of manually checking materials for big builds, so this app handles it for me.

## Built with
- Java 21
- JavaFX
- Gradle
